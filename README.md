This is a **reorganized** implementation of [Informer](https://arxiv.org/2012.07436) based on [the official implementation](https://github.com/zhouhaoyi/Informer2020) and PyTorch Lightning.

## Requirements

* numpy
* pandas
* scikit-learn
* torch
* torchmetrics>=0.3.0
* pytorch-lightning>=1.3.0
